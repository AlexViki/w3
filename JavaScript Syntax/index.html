JavaScript and Camel Case

programmers have used different ways of joining multiple words into one variable name

  • Hyphens: first-name, last-name, master-card. Hyphens are not allowed in JavaScript;
  • Underscore: first_name, last_name, master_card;
  • Upper Camel Case (Pascal Case): FirstName, LastName, MasterCard; click SHOW
  • Lower Camel Case: firstName, lastName, masterCard
