JavaScript Break and Continue

The break statement "jumps out" of a loop.
The continue statement "jumps over" one iteration in the loop.

The Break Statement

You have already seen the break statement. It was used to "jump out" of a switch() statement. The break statement can also be used to jump out of a loop.
The break statement breaks the loop and continues executing the code after the loop (if any):

The Continue Statement

The continue statement breaks one iteration (in the loop), if a specified condition occurs, and continues with the next iteration in the loop.

JavaScript Labels

To label JavaScript statements you precede the statements with a label name and a colon (see javascript file)
The break and the continue statements are the only JavaScript statements that can "jump out of" a code block.
The continue statement (with or without a label reference) can only be used to skip one loop iteration.
The break statement, without a label reference, can only be used to jump out of a loop or a switch.
With a label reference, the break statement can be used to jump out of any code block

A code block is a block of code between { and }.
